# LeadrBoard Backend

A FastAPI-based REST API for LeadrBoard, a social platform for gamers to discover games, write reviews, and share their gaming experiences with friends.

## Overview

LeadrBoard connects gamers through game reviews and social interactions. Users can browse games from the IGDB database, write detailed reviews with ratings and playtime, follow other gamers, and interact through likes and comments on reviews. The platform features a personalized activity feed showing reviews from followed users.
